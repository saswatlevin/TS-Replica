In all Create functions, check if the object already exists.
We need the following pieces of data
1) userId -> From URL Parameter | From Session
2) product_id -> From custom property created on request body | From front end
4) product_body -> From request body

createProduct -> 
  * Get a DEEP CLONE of the request body

  * Check if the request body is empty, if so, throw an EmptyRequestBodyError

  * Check if the product already exists, if yes, throw a DuplicateDocumentRequestErro

  * Generate the product_id as a 12-character random string.

  * Generate the image_id.

  * Generate the sku.

  * Generate the doctype as PRODUCT.

  * Add the sku to the product_items key-value pair in the DEEP CLONE of the request body.

  * Add the image_id to the product_images key-value pair in the DEEP CLONE of the request body.

  * Add the product_id and doctype to the DEEP CLONE of the request_body. This is the product.

  * Use the Model.create() function to create the product by inserting the request_body.
