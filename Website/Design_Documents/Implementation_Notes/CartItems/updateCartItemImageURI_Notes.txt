We need the following pieces of data:
* product_id -> Request body | From Frontend

updateCartItemImageURI ->
* Check if the user exists or not, if not, throw a ResourceNotFoundError.
* Create the filter: {"CartItems.product_id": product_id}.
* Create the update_object as update_object = {
            $set: {
                "CartItems.$[item].cart_item_image_uri": updated_cart_item_image_uri
            }
        };

* Use updateMany() to update the cart_item_uri of that cart item.
* Return the result.

NOTE: updateCartItemImageURI is an express HELPER FUNCTION and not a middleware.

NOTE: We do not check if the updated cart_item_image_uri value already exists since we aim for database operations to be atomic and updateProductImgeURI already checks if the image_uri value exists.


NOTE: updateMany() is used to update the image_uri of all CartItems of the same product across all users. 

NOTE: No need to check if product exists, since updateProductName already checks that and in the future DB operations will be made atomic.
