* user_id -> User params | From session
* cart_item_id -> User params | From session

* Get the request body.
* Check if the request_body is empty or not, if yes, throw an EmptyRequestBodyError.
* Check if the user exists or not, if not, throw a ResourceNotFoundError.
* Check if the cart item exists or not, if not, throw a ResourceNotFoundError.
* Create the filter: {user_id: user_id, cart_item_id: cart_item_id}.
* Create a Query object from the request_body which contains {'CartItems.$.cart_item_qantity': value}
