We need the following pieces of data:
  * user_id      -> Session
  * product_id   -> Frontend
  * review_id    -> Frontend

deleteReview ->
  * Check if the request_body is empty. If so, throw an EmptyRequestBodyError.
  * Get the user_id from req.params.
  
  * Check if the user exists or not. If it doesn't, throw a ResourceNotFoundError.
  * Get the product_id from the req.body.

  * Check if the product exists. If it does not, then throw a ResourceNotFoundError.
  * Get the review_id from the request_body.

  * Check if the Review to be deleted exists. If it does not, throw a ResourceNotFoundError.

  * Create the delete_query as {user_id: user_id, review_id: review_id, product_id: product_id}.
  
  * Call findoneAndDelete() with the delete_query.
  
  * Send the result back with a status of 200 and as JSON.
