We need the following pieces of data:
* user_id -> User params | From session
* cart_item_id -> User params | From session

updateCartItemURI ->
* Check if the user exists or not, if not, throw a ResourceNotFoundError.
* Check if the cart item exists or not, if not, throw a ResourceNotFoundError.
* Check if the updated cart_item_uri value already exists, if so, throw a RedundantUpdateError.
* Create the filter: {user_id: user_id, cart_item_id: cart_item_id}.
* Create a Query object from the request_body which contains {'CartItems.$.cart_item_uri': value}
* Use findOneAndUpdate() to update the cart_item_uri of that cart item.
* Return the result.

NOTE: updateCartItemURI is an express HELPER FUNCTION and not a middleware.
