============CREATE SHIPPING ADDRESS REQUEST SCHEMA============
address_type_id -> length(1), required() -> A single-character string that takes a digit.

company_name -> max(100), optional() -> A company_name can have letters, spaces, commas and dashes.

address -> min(1), max(200), required() -> An address can only have letters, numbers, dashes, spaces and commas.

apartment -> max(100), optional() -> An apartment can have letters, numbers, spaces and brackets, i.e., ( and ).

city -> min(3), max(100), required() -> A city can have letters, spaces, dashes and single quotes.

administrative_division -> max(100), optional() -> An administrative_division can only have letters, spaces, single-quotes and dashes. 

country -> required() -> A country can only be one of the following: "United States", "United Kingdom", "Canada", "Australia", "New Zealand", "Ireland", "Singapore", "Hong Kong", "Japan".

postal_area -> min(3), max(10), required() -> A postal_area can only be a valid postal_address.

phone_number -> min(8), max(12), required() -> A phone_number must be valid and with country code.
