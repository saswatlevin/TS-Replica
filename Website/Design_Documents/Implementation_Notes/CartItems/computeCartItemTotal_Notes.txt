We need the following pieces of data:
user_id -> request_params | Session

computeCartItemTotal ->
  * Check whether the request body is empty. If so, throw an EmptyRequestBodyError.
  * Check if the user exists or not. If it does not, throw a ResourceNotFoundError.
  * Create a filter of {user_id: user_id}.
  * There is no need to create a query object since we will be getting all the CartItems in the User document.
  * Calculate the total_cart_item_price by multiplying the quantity into the price of each cart_item.
  * Create an update_query with {total_cart_item_price: total_cart_item_price}
  * Update the total_cart_item_price in the total_cart_item_price field in the respective User document by using findOneAndUpdate(filter, update_query).lean().
  * return the total_cart_item_price.

SPECIAL NOTE: The computeCartItemTotal() function will be called in the following controllers:
  * createCartItem()
  * deleteCartItem()
  * updateProductPrice()
  * updateCartItemQuantity()
* This function will be a HELPER FUNCTION.
