We need the following pieces of data:
* product_id -> Request body | From frontend

updateCartItemPrice ->
  * Get the product_id from req.body.
  * Get the updated_product_price from res.locals.
  * Deploy the buildUpdateCartItemPricePipeline aggregation pipeline. This pipeline will update the item_total, discount_amount and discounted_total for all the CartItems that belong to the Product with the given product_id across all Users.

NOTE: We use updateMany() to call the buildUpdateCartItemPricePipeline aggregation pipeline.
NOTE: No need to check if product exists, since updateProductPrice already checks that and in the future DB operations will be made atomic.
