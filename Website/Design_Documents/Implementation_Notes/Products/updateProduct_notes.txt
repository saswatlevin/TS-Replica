* Get the product_id from the request params.

* Check if the Product document exists or not, if not, throw a ResourceNotFoundError.

* CHECK IF THE DATA THAT YOU'RE ABOUT TO UPDATE ALREADY EXISTS IN THE DATABASE -> If yes, throw CustomError with error code 400.
* Get a DEEP CLONE of the request_body.

* Create a filter, {product_id: product_id}.
* Use the findOneAndUpdate() function with the filter, the DEEP CLONE of the request_body, {runValidators: true} and chain it with .lean() to update the product price.

* Get the result from findOneAndUpdate().
* Send the result back as JSOn with a status code of 200.
